DEFI ENDURANCE


La circonscription de Beauvais Nord qui regroupe plusieurs écoles souhaite mettre à disposition des enseignants une application ludique et pédagogique pour les défis sportifs.

Ces défis sportifs ont lieu tous les ans à une période précise. Chaque classe doit définir un objectif à atteindre.

En fonction de cet objectif, les élèves courent et l’enseignant comptabilise la distance totale parcourue (la somme de chaque parcours d’élèves) pendant une séance.
L’enseignant peut réaliser autant de séances qu’il le souhaite pour réaliser l’objectif pendant la période définie.

  • Date: Novembre 2017

MAC OS VIRTUALISATION


Ce projet était de créer une machine virtuel macOS ,via au système d'exploitation Sierra.

Pour cela j'avais utilisé le logiciel de virtualisation VMware car cette une version est stable pour créer des machines mac par rapport à son voisin VirtualBox.

J'au dû installer MacOs Sierra à travers une image VMware et configurer la machiner puis j'ai rajouter un code au fichier config de mac avant le premier lancement de la machine afin de pourvoir démarrer.

Ensuite une fois que la machine à démarrer et a été configurer j'ai installer sur la machine Mac, VMware Tools afin d'améliorer la fluidité de la machine

  • Date: Mars 2018
